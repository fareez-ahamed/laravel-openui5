## Laravel-OpenUI5

A sample application using Laravel and OpenUI5

###Requirements
- PHP 5.4
- MySQL 5.5
- Composer
- Bower
- Gulp

###Steps to Run
- `composer install`
- Update the database connection settings in `app/config/database.php`
- `php artisan migrate`
- `php artisan db:seed`
- `bower install`
- `npm install'
- `gulp move-openui5`
